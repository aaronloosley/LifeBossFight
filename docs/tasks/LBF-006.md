# LBF-006: Password reset and email verification flow

**Tag:** LBF-006  
**Category:** Authentication

## Description
Users need reliable self-service recovery and account verification before the product can be trusted in production. This task covers reset requests, verification links, and related UX.

## Steps
1. Add a forgot-password route and request form.
2. Customize Firebase email templates for reset and verification flows.
3. Handle success and error responses without leaking account existence.
4. Gate privileged app actions if email verification becomes mandatory.
