# LBF-057: Package Missing mission design and implementation

**Tag:** LBF-057  
**Category:** Mission Content

## Description
Create a consumer-problem mission for missing deliveries, proof-of-delivery disputes, and merchant/carrier escalation. This is simple to understand, widely relatable, and ideal for acquisition content.

## Steps
1. Define steps for checking tracking, safe places, neighbours, and delivery proof.
2. Add branches for retailer contact, courier claim, chargeback preparation, and refund evidence.
3. Include prompts for screenshots, timestamps, and doorstep photos where available.
4. Verify that the report summary is concise enough for customer support use.
