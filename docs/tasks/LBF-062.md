# LBF-062: Bank Fraud Charge mission design and implementation

**Tag:** LBF-062  
**Category:** Mission Content

## Description
Add a financial-emergency mission for disputed charges, compromised cards, and fraud follow-up. This is both high-trust and high-repeat because users often feel overwhelmed by the sequence.

## Steps
1. Define the safest order for freezing cards, contacting the bank, documenting charges, and preserving messages.
2. Add steps for police reports if relevant, merchant follow-up, and identity-protection actions.
3. Include prompts for timestamps, screenshots, and reference numbers.
4. Review the mission to avoid overpromising reimbursement outcomes.
