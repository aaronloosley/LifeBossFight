# LBF-067: Scam Recovery mission design and implementation

**Tag:** LBF-067  
**Category:** Mission Content

## Description
Add a recovery mission for online scams, impersonation, and fraudulent transfers. This is highly relevant, emotionally charged, and valuable for repeat trust.

## Steps
1. Define the first-hour response steps for bank contact, account security, evidence preservation, and reporting.
2. Add branches for marketplace scams, impersonation, romance scams, and fake support attacks.
3. Include prompts for screenshots, payment references, and device/account cleanup.
4. Review the mission with strong safety boundaries around legal and financial claims.
