# LBF-051: Burst Pipe production content polish

**Tag:** LBF-051  
**Category:** Mission Content

## Description
Use the existing Burst Pipe mission as the production benchmark for future bosses. Tightening the current mission first creates a reusable standard for pacing, evidence prompts, escalation advice, and report quality.

## Steps
1. Review the current Burst Pipe content for clarity, completeness, and legal/safety caveats.
2. Refine step order, evidence prompts, timeline guidance, and summary output based on real-user testing.
3. Add edge cases such as landlord involvement, insurance contact, and emergency shutoff branching.
4. Document the quality bar that every new mission must meet before release.
