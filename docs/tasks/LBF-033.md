# LBF-033: Deployment rollback and release orchestration

**Tag:** LBF-033  
**Category:** CI/CD

## Description
Production launches need a controlled release process with rollback paths, owners, and communication steps. This lowers the blast radius of bad deployments.

## Steps
1. Define who can deploy and who approves production releases.
2. Create a rollback checklist covering code, config, and Firebase rule changes.
3. Add release notes and deployment verification steps.
4. Practice a rollback in a safe environment before launch.
