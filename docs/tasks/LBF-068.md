# LBF-068: Tradesperson Ghosted mission design and implementation

**Tag:** LBF-068  
**Category:** Mission Content

## Description
Create a mission for deposits paid, work incomplete, or repeated no-shows by tradespeople. This fits the Home Chaos world and is instantly relatable for short-form content.

## Steps
1. Map evidence gathering for quotes, invoices, messages, missed appointments, and unfinished work.
2. Add branches for refund requests, complaint escalation, card disputes, and replacement contractors.
3. Include prompts for photos, deadlines, and communication logs.
4. Test the output for usefulness in both negotiation and formal complaint contexts.
