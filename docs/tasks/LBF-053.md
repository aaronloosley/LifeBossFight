# LBF-053: Lost Passport Abroad mission design and implementation

**Tag:** LBF-053  
**Category:** Mission Content

## Description
Add a travel-crisis mission for users who lose a passport overseas and need an exact order of actions. This content is both highly viral and practically valuable for travelers.

## Steps
1. Map the mission from identity confirmation and embassy contact through temporary travel document recovery.
2. Include steps for police reports, airline coordination, accommodation extensions, and backup identity evidence.
3. Document region-specific caveats and the need to verify embassy guidance.
4. QA the mission for offline usefulness and minimal-assumption wording.
