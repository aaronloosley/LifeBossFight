# LBF-039: Data retention and deletion lifecycle

**Tag:** LBF-039  
**Category:** Compliance

## Description
Decide how long user accounts, evidence, and operational logs are retained, and how deletion requests are fulfilled. Retention policy is both a privacy and a cost-management concern.

## Steps
1. Define retention periods for each major data class.
2. Add account deletion and evidence deletion workflows as needed.
3. Ensure backups and logs are covered by the retention policy.
4. Document how deletion requests are verified and executed.
