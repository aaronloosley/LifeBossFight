# LBF-042: Localization QA and content review process

**Tag:** LBF-042  
**Category:** Internationalization

## Description
If multiple locales will ship, translations need quality control and layout validation. This task ensures localized experiences remain usable and trustworthy.

## Steps
1. Define review ownership for translated content.
2. Check translated UI for truncation, overflow, and tone consistency.
3. Validate locale-specific dates, times, and number formatting.
4. Add locale coverage to regression QA and release checklists.
