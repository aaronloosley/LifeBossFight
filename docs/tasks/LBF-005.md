# LBF-005: Email/password login, logout, and session restore

**Tag:** LBF-005  
**Category:** Authentication

## Description
Implement a full email/password sign-in flow and ensure sessions persist correctly across refreshes and tab restores. Without this, a production auth model remains incomplete and fragile.

## Steps
1. Add login form submission handlers backed by Firebase Auth.
2. Restore authenticated state on refresh with a shared auth provider.
3. Add explicit logout controls and redirect behavior.
4. Test sign-in, sign-out, and refresh continuity across core pages.
