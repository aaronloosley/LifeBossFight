# LBF-028: Lighthouse and Core Web Vitals optimization

**Tag:** LBF-028  
**Category:** Performance

## Description
Improve user-perceived speed and technical SEO quality by addressing Core Web Vitals and Lighthouse issues. This is especially important for mobile-first usage on unstable networks.

## Steps
1. Run Lighthouse audits on representative routes.
2. Fix opportunities related to image sizing, unused code, and layout stability.
3. Monitor CLS, LCP, INP, and TTFB in preview and production.
4. Document acceptable thresholds in the release checklist.
