# LBF-010: User profile persistence and onboarding state

**Tag:** LBF-010  
**Category:** Data

## Description
Store an application-level user profile so Life Boss Fight can track names, avatars, preferences, and onboarding milestones independent of raw auth records. This creates a durable foundation for personalization and support.

## Steps
1. Define the `users/{userId}` document shape.
2. Create or merge a profile record after first sign-in.
3. Track onboarding completion, notification preferences, and support metadata.
4. Add tests that verify profile creation and update behavior.
