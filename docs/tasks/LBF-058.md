# LBF-058: Subscription Won’t Cancel mission design and implementation

**Tag:** LBF-058  
**Category:** Mission Content

## Description
Add a mission for users trapped in difficult cancellation flows or recurring charges. This boss is consumer-friendly, relatable, and useful for trust-building content.

## Steps
1. Map the order of actions across in-app cancellation, web account settings, support contact, and bank escalation.
2. Add evidence prompts for terms, screenshots, billing cycles, and cancellation attempts.
3. Include safe language around chargebacks and complaint escalation.
4. Test the mission against both straightforward and intentionally hostile cancellation patterns.
