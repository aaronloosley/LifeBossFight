# LBF-061: Boiler Died mission design and implementation

**Tag:** LBF-061  
**Category:** Mission Content

## Description
Create a home-emergency mission for heating or hot-water failure, especially during cold weather. This boss fits the strong Home Chaos world and has clear urgency and referral value.

## Steps
1. Map the immediate checks, safety warnings, and evidence prompts for outage timing and temperature impact.
2. Add branches for tenants, homeowners, emergency engineers, and vulnerable household members.
3. Include guidance on repair chasing, alternative heating evidence, and reimbursement records.
4. QA the mission against both owner-occupier and rental scenarios.
