# LBF-017: OWASP-aligned application security review

**Tag:** LBF-017  
**Category:** Security

## Description
Run a structured security review covering authentication, authorization, data exposure, client-side storage, dependency risk, and abuse scenarios. Production readiness requires more than just passing happy-path tests.

## Steps
1. Assess the app against relevant OWASP ASVS or Top 10 guidance.
2. Document findings, severity, and remediation owners.
3. Close critical and high-risk issues before launch.
4. Schedule follow-up reviews after major architectural changes.
