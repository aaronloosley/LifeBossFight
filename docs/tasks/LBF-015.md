# LBF-015: Environment variable and secret management

**Tag:** LBF-015  
**Category:** Security

## Description
Document and harden how secrets and public configuration are managed across local development, CI, preview, and production. Poor secret hygiene creates avoidable outages and security incidents.

## Steps
1. Classify which values are public Firebase config and which are true secrets.
2. Store secrets only in approved secret managers and hosting dashboards.
3. Rotate any shared credentials that were previously distributed insecurely.
4. Document onboarding steps for developers without exposing sensitive values.
