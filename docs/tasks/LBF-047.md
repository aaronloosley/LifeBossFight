# LBF-047: Admin operations and support tooling

**Tag:** LBF-047  
**Category:** Operations

## Description
Determine what internal tools or dashboards the team needs to inspect account state, troubleshoot issues, and handle support cases safely. Manual console-driven operations do not scale well.

## Steps
1. List recurring support and operational actions that require internal visibility.
2. Decide whether admin capabilities live in Firebase console, custom dashboards, or scripts.
3. Design access controls and audit logging for admin actions.
4. Document safe operating procedures for internal tools.
