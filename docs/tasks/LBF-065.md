# LBF-065: Funeral Admin mission design and implementation

**Tag:** LBF-065  
**Category:** Mission Content

## Description
Add a grief-support mission focused on practical funeral and estate administration tasks. This boss should reduce overwhelm during a highly emotional period by turning chaos into a paced checklist.

## Steps
1. Define step groups for immediate notifications, document gathering, service coordination, and follow-up admin.
2. Add respectful wording and optional branching for cultural or family decision differences.
3. Include prompts for contact logs, required certificates, and key deadlines.
4. Test with sensitivity reviewers to ensure the tone remains supportive and clear.
