# LBF-064: Breakup Logistics mission design and implementation

**Tag:** LBF-064  
**Category:** Mission Content

## Description
Create a life-admin mission for the practical steps after a breakup, including belongings, bills, accounts, and boundary-setting records. This is emotionally charged but also deeply useful.

## Steps
1. Map the mission around immediate stabilisation, shared-account review, item inventory, and communication planning.
2. Add branches for cohabitation, pets, subscriptions, and digital account separation.
3. Keep tone calm, non-judgmental, and safety-aware.
4. Review the mission for sensitive content handling and escalation boundaries.
