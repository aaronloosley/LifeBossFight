# LBF-059: Flight Cancelled mission design and implementation

**Tag:** LBF-059  
**Category:** Mission Content

## Description
Create a travel disruption mission that helps users recover options, evidence, and compensation records when a flight is cancelled. This is highly visual, urgent, and broadly understandable.

## Steps
1. Outline the immediate sequence for airline desk contact, app evidence capture, rebooking options, and accommodation planning.
2. Add branches for compensation eligibility, multi-leg trips, and travel insurance.
3. Include offline-friendly prompts for screenshots and receipts while networks are overloaded.
4. Review the mission for cross-border wording and airline-policy variability.
