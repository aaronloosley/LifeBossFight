# LBF-055: Landlord Repair Fight mission design and implementation

**Tag:** LBF-055  
**Category:** Mission Content

## Description
Build a mission for unresolved repair disputes with landlords or property managers. This is a strong repeat-traffic scenario because it combines evidence gathering, escalation, and deadline tracking.

## Steps
1. Define the evidence pack for defects, timelines, communications, and habitability impact.
2. Create step branches for emergency repairs, repeated chasers, and formal complaint escalation.
3. Include reminders for written records, photos, and local tenancy guidance.
4. Validate that the report output helps users escalate clearly and safely.
