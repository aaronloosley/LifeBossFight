# LBF-026: End-to-end coverage for account and mission flows

**Tag:** LBF-026  
**Category:** Testing

## Description
Expand Playwright coverage so the most valuable user journeys are proven in a browser. Production confidence depends on testing the app the way users actually experience it.

## Steps
1. Cover registration, login, logout, password reset, and demo fallback flows.
2. Cover starting, resuming, and completing a mission run.
3. Add evidence capture, summary, and timeline assertions.
4. Run E2E tests against stable preview or emulator-backed environments.
