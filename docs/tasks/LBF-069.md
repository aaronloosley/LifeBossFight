# LBF-069: House Move Chaos mission design and implementation

**Tag:** LBF-069  
**Category:** Mission Content

## Description
Add a mission for the multi-step mess of moving house, including utilities, address changes, inventory, and dispute prevention. This is broad and useful even when no single dramatic incident occurs.

## Steps
1. Break the mission into pre-move, moving-day, and post-move phases.
2. Add checklists for utilities, inventory photos, key handover, and service transfers.
3. Include branches for delayed movers, damaged items, and missed address updates.
4. Validate that the mission balances planning support with incident-response usefulness.
