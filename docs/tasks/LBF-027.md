# LBF-027: Performance budgets and profiling workflow

**Tag:** LBF-027  
**Category:** Performance

## Description
Define acceptable performance targets and measure the app against them so performance work is objective and repeatable. Without budgets, regressions creep in unnoticed.

## Steps
1. Set page-load and interaction targets for mobile and desktop.
2. Profile the login page, mission screens, and evidence workflows.
3. Document heavy bundles, expensive renders, and oversized assets.
4. Create follow-up tickets for the most impactful improvements.
