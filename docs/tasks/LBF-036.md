# LBF-036: Analytics and product instrumentation plan

**Tag:** LBF-036  
**Category:** Analytics

## Description
Define what user behavior and funnel data should be captured so product decisions are evidence-based after launch. Instrumentation should be privacy-conscious and aligned with explicit goals.

## Steps
1. List the key questions product and operations want analytics to answer.
2. Define events for onboarding, sign-in, mission launch, completion, and evidence usage.
3. Add consistent event naming and schema documentation.
4. Validate consent and privacy requirements before turning analytics on.
