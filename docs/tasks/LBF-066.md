# LBF-066: Pet Emergency mission design and implementation

**Tag:** LBF-066  
**Category:** Mission Content

## Description
Create a mission for urgent pet-health incidents where owners need a calm order of actions, evidence, and care decisions. This boss has high emotional stakes and strong broad appeal.

## Steps
1. Map first actions for emergency vet contact, symptom timing, transport preparation, and cost notes.
2. Add branches for poisoning, injury, missing pet insurance details, and aftercare follow-up.
3. Include clear prompts for photos, medication history, and vet advice records.
4. Validate that the mission stays within safe non-diagnostic boundaries.
