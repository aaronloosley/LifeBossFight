# LBF-063: Parking Fine Challenge mission design and implementation

**Tag:** LBF-063  
**Category:** Mission Content

## Description
Build a mission for contesting parking fines or penalty notices with evidence and deadlines. This is a compact, understandable, and highly actionable boss.

## Steps
1. Outline steps for checking notice details, signage evidence, time records, and appeal windows.
2. Add branches for private parking operators versus public authority penalties.
3. Generate a report structure suitable for a concise appeal submission.
4. Test the mission with varied evidence quality and time-pressure scenarios.
