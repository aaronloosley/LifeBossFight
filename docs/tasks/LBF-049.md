# LBF-049: Launch rehearsal and go-live checklist

**Tag:** LBF-049  
**Category:** Release Management

## Description
Run a final staged rehearsal so the team can validate launch readiness before exposing the app broadly. Rehearsals surface missing steps that ordinary development work rarely catches.

## Steps
1. Create a detailed go-live checklist covering code, config, legal, support, and monitoring.
2. Rehearse the release in staging or preview using real operators.
3. Record any surprises, missing approvals, or rollback gaps.
4. Update the final launch checklist based on rehearsal outcomes.
