# LBF-043: Browser, device, and platform compatibility matrix

**Tag:** LBF-043  
**Category:** Quality Assurance

## Description
Document which browsers, operating systems, and device classes are officially supported so QA and support have clear expectations. This avoids hidden platform-specific failures late in launch.

## Steps
1. Define minimum supported browser and OS versions.
2. Test critical flows on the approved matrix.
3. Identify degraded experiences and document them clearly.
4. Update support documentation with any known limitations.
