# LBF-001: Production readiness gap assessment

**Tag:** LBF-001  
**Category:** Program Management

## Description
Audit the current MVP against launch expectations so the team knows exactly which blockers exist before scheduling release work. This task should produce a ranked readiness matrix covering product, engineering, security, compliance, support, and operational risks.

## Steps
1. Inventory the current application features, deployment shape, and known limitations.
2. Define launch criteria for MVP, beta, and production milestones.
3. Document blockers, owners, dependencies, and recommended sequencing.
4. Review the assessment with product and engineering stakeholders and turn it into the working roadmap.
