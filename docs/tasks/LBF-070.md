# LBF-070: Deposit Dispute mission design and implementation

**Tag:** LBF-070  
**Category:** Mission Content

## Description
Build a mission for recovering rental deposits or preparing a dispute response. This has strong repeat-traffic value and pairs naturally with landlord and moving workflows.

## Steps
1. Define evidence requirements for inventory, cleaning, repairs, deductions, and move-out communication.
2. Add branching for protected deposit schemes, informal rentals, and partial refund offers.
3. Generate a concise dispute pack with chronology and supporting evidence references.
4. QA the mission for fairness, clarity, and deadline awareness.
