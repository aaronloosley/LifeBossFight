# LBF-021: Responsive UI polish across core flows

**Tag:** LBF-021  
**Category:** UX

## Description
Validate and refine the mobile-first interface across common breakpoints so the app feels intentional instead of merely functional. This includes auth screens, mission runners, evidence capture, and settings.

## Steps
1. Review layouts on small phones, large phones, tablets, and desktop widths.
2. Fix overflow, spacing, tap target, and safe-area issues.
3. Improve typography hierarchy and visual consistency across pages.
4. Capture before-and-after screenshots for design review.
