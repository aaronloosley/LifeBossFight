# LBF-014: Storage and Realtime Database rules review

**Tag:** LBF-014  
**Category:** Security

## Description
If the product uses Firebase Storage or Realtime Database, those services need explicit access controls just as much as Firestore. Leaving defaults or permissive rules in place is a launch blocker.

## Steps
1. Identify every Firebase data service used in each environment.
2. Write user-scoped Storage rules for uploaded evidence and report assets.
3. Add Realtime Database rules only if that service is adopted.
4. Test unauthorized access attempts before launch.
